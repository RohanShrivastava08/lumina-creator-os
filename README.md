## ✦ Core Philosophy

Lumina is not a "caption generator." It is a synthesis engine. 

In a world of digital noise, Lumina prioritizes **intentionality, clarity, and precision**. Every artifact generated is architected with platform-specific fidelity, ensuring your narrative resonance is maximized while maintaining a low-friction, luxury editorial aesthetic.

## ✺ Primary Features

### 1. Neural Ghostwriter
Manifest professional content drafts in seconds. Our engine understands the nuance between a provocative LinkedIn thought-leadership piece, an emotional Instagram story, and a high-signal X thread.

### 2. Multi-Vector Platform Logic
*   **LinkedIn**: Professional provocation & authority building.

Lumina doesn't just write; it visualizes. Every generation includes cinematic, highly-detailed image prompts optimized for:
*   **Midjourney v6.0**

Real-time synthesis reports providing scores on:

A unique visual logic where the "Artifact" (your content) exists in a high-contrast inverted space, separating the **work** from the **interface**.

## ❖ Deployment & Initialization

### Environment Configuration
Lumina OS requires a valid Neural Link to the Google Generative AI matrix. Define your API key in the environment:

```env
# .env
GEMINI_API_KEY=your_neural_link_key
```

### Installation
```bash
npm install
npm run dev
```
